Generate AI-powered blog posts instantly using the Gemini API with a clean, modern WordPress interface.

== Description ==
AutoBlogGen allows you to generate high-quality AI-driven blog posts using the Gemini API.  
Users can securely verify their API key and create instant, SEO-friendly content directly from prompts — perfect for writers, marketers, and bloggers.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/autobloggen/`.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Use the `[ai_blog_generator]` shortcode on any page or post to display the AI blog generator form.

== Frequently Asked Questions ==
= Does this plugin store my API key? =
No, your Gemini API key is stored only temporarily in the session and is never saved in the database.

= Do I need a Gemini API key? =
Yes. Obtain your key from the Google AI Gemini developer console.

= Is this plugin compatible with all themes? =
Yes, it’s designed to be fully responsive and integrate with any modern WordPress theme.
