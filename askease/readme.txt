A modern FAQ manager with frontend admin login, and Add/Edit/Delete support using modal popups.

== Installation ==
1. Upload `askease` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Use the shortcode `[askease_faq]` in any page.
